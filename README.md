# Algshop

منصة احترافية لعرض المنتجات للبيع والشراء. تعمل بـ Next.js 16 + Firebase (المستخدمين) + Cloudflare R2 (الصور) وجاهزة للنشر على Vercel.

## المميزات

- واجهة عربية RTL احترافية مستوحاة من أكبر المتاجر.
- إنشاء حساب وتسجيل دخول عبر Firebase (email/password + Google).
- إضافة منتجات مع صور، وصف، فئة، حالة، سعر قبل التخفيض، وخيارات اختيارية.
- رفع الصور إلى Cloudflare R2 عبر روابط مؤقتة (presigned URLs).
- بحث وفلاتر حسب الفئة والحالة والسعر.
- صفحة تفاصيل المنتج مع معرض صور وطلب شراء.
- لوحة تحكم البائع: منتجاتي، مبيعاتي، طلباتي.
- وضع تجريبي يعمل محلياً بدون Firebase.

## التشغيل المحلي

```bash
npm install
npm run dev
```

بدون إعداد Firebase يعمل المشروع في الوضع التجريبي ( demo ) تلقائياً مع بيانات وهمية. لإعداد الإنتاج، انسخ `.env.example` إلى `.env.local` واملأ المتغيرات.

## النشر على Vercel

1. اربط المستودع بمشروع Vercel.
2. أضف متغيرات البيئة من `.env.example` في لوحة تحكم Vercel.
3. اضبط CORS على bucket R2 للسماح بالـ PUT من نطاق الموقع.
4. انشر.

## هيكل المشروع

- `app/` صفحات Next.js App Router.
- `components/` المكونات المشتركة.
- `lib/` إعدادات Firebase، Cloudflare R2، وطبقة الخدمات.
- `types/` أنواع TypeScript.
